# 基本指令

## 設定

設定識別資料
```
git config --global user.name "samchentw"
git config --global user.email penguin.bear11290330@gmail.com
```

## 檢查讀者的設定
```
git config --list
```


## 初始化
```
git init
```

## 查看狀態
```
git status
```

## 將檔案加入到索引
```
git add index.txt  //加入單一
git add .          //加入全部
```


## 提交
```
git commit -m "這是說明"
```

## 查看版本
```
git log
```
